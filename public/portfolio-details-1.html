What this Project Accomplishes

Agricultural productivity is often threatened by crop infections. This project engineered an end-to-end computer-vision mobile application providing rapid agricultural crop diagnosis via image-processing neural networks.

The app allows farmers to capture leaves in real-time. By deploying edge-compiled TensorFlow/Firebase ML models directly onto Android devices, we bypassed server dependency and network latencies.

Key Features & Implementation

  • Offline Inference Engine: Integrated Firebase ML Kit on-device model interpreters, delivering rapid crop diagnosis under 150ms directly on edge hardware.

  • Mobile Vision Framework: Structured Android Studio Java codebase implementing camera stream capture, buffer conversions, and overlay displays.

  • Neural Network training: Trained custom convolutional neural networks (CNNs) in Python on plant datasets to accurately identify leaf spot, blight, and rust conditions.

  • Scholarly Impact: Co-authored and published structural research documentation detailing classification accuracies and inference statistics in the IJSREM journal.
